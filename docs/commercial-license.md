## 4. Plain English FAQ

### Q1: Can I use CertifyAI for my startup's production LLM?

**Yes.** Both the Pro and Enterprise licenses allow commercial use. You can run CertifyAI against your production LLM endpoints, generate compliance evidence, and share reports with auditors and customers. No additional licensing is needed.

### Q2: Can I resell the tool itself to my clients?

**No.** No tier allows you to resell, sublicense, or redistribute the CertifyAI software itself. The Enterprise license lets you use the tool to serve clients (generate reports for them), but you cannot give them a copy of the tool. If a client wants their own copy, they need to purchase their own license.

### Q3: Can I modify the attack scenarios and keep them proprietary?

**Free/Pro:** The attack scenarios are part of the Software. Modification requires source code access, which is only available in the Enterprise tier. You can modify configuration files and compliance framework YAML mappings without source access.

**Enterprise:** Yes. You have source access and can modify attack scenarios. You may keep your modifications proprietary as long as they are used only in your client work and not distributed as a competing product.

### Q4: Do I need to attribute CertifyAI in my compliance reports?

**Free (Apache 2.0):** Yes, attribution is required per the Apache 2.0 license.

**Pro:** No attribution required.

**Enterprise:** No attribution required. You may white-label reports with your own branding.

### Q5: What happens if I stop paying for updates?

The Software continues to work indefinitely at the version you have. You don't lose access to features you already have. You simply stop receiving new attack scenarios, bug fixes, and feature updates. The Pro license includes 12 months of updates; after that, you can choose to renew or continue with your current version.

### Q6: Can I use CertifyAI for government work?

**Yes.** There are no restrictions on using the Software for government or public sector work. Enterprise licensees serving government clients should ensure their modifications comply with applicable government procurement and security requirements.

### Q7: Is the Web Dashboard included under the same license?

**Yes.** The Web Dashboard is part of the Software and is covered by the same license terms as the CLI and TUI. Pro and Enterprise licenses include the Web Dashboard. The Free tier (Apache 2.0) does not include the Web Dashboard.

### Q8: Can I run CertifyAI in my CI/CD pipeline?

**Yes.** The CLI is designed for non-interactive use. You can run `certifyai run` in GitHub Actions, GitLab CI, Jenkins, or any CI/CD system. The license covers this use case. Exit codes and JSON output make it CI-friendly.

### Q9: What data does CertifyAI send to your servers?

**None.** CertifyAI has no telemetry, no phone-home, no data collection. It runs entirely on your machine. The only outbound network calls are the LLM API requests you configure it to make (e.g., to OpenAI or your Ollama instance). We never see your prompts, responses, or compliance data.

### Q10: Can I get a refund if CertifyAI doesn't work for me?

**We follow Gumroad's standard refund policy.** If the Software has a material defect that prevents its intended use, we will fix it or provide a refund. We cannot refund purchases where the Software was functional but didn't meet unstated expectations — please evaluate the Free tier thoroughly before purchasing.
